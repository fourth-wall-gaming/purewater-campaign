# Setting Sources -- Purewater

Canonical worldbook sources for **And Then the Dragons Came: Purewater**, a faithful
Classic Fantasy Imperative (CFI) port of the original *And then the Dragons Came*
campaign (originally set in Zorastor, world of Greymoor). The `lore/` directory at
the repo root contains this material sliced into database-ready entries; the files
here are the fuller source documents, plus a synthesized narrative of the opening arc.

| File | Contents | Audience |
|---|---|---|
| `the-story-so-far.md` | Prose synthesis of the canonical opening (novelization ch. 1-16) | player + GM |
| `conversion-from-zorastor.md` | How the original Zorastor campaign maps to Purewater | GM |
| `setting-exhaustive.md` | Full setting reference: geography, factions, magic, plot, themes | GM |
| `magic.md` | The five-tradition Mythras magic of Purewater (mapped to CFI in lore/) | GM |
| `organizations.md` | The city's organizations: nature, real-vs-stated aims, ranks, leaders | GM |
| `player-organizations.md` | Player-facing organization guide | player |
| `characters-source.md` | Source character notes | GM |
| `adventure.md` | The adventure structure and beats | GM only |

The novelization (`And then the Dragons Came.pdf`) is the canonical prose source for
the story, tone, and dramatis personae; the events of its sixteen chapters are also
captured as journal entries in `journal/events.json`.
